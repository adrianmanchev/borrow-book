# Borrow book

List members, books and members' borrowing of books for a library.

## Usage

### Installation

```
npm install
npx sequelize-cli db:create
```

### Run

```
node index.js
```

## Features

- List members
- Get member book detail
- Create a new member
- List books
- List book detail
- Create a new book
- Borrow and return book
